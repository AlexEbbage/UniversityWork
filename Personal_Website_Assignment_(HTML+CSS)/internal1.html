Shape Shooter

Shape Shooter is the first game I made in Java. It is a simple twin-stick style shooter game, which uses simple colours and shapes for the graphics. It took me about 2 months to complete and taught me a lot. The player controls the ship using the arrow keys and mouse. There is a simple menu and highscore system which I added to make the game feel more complete. There are 3 enemies in the game, each with their own simple behaviours. There is also a particle effects system in the game, which I found very enjoyable to make.

DnD-Style Project

I started this project because I wanted to make it more convenient for my friends and I to play DnD whilst attending different Universities. It's been a steep learning curve but I'm loving it. I planned this project much better than my previous one, so features can be added easily and code isn't difficult to come back to when I have breaks from study. I am very proud of the lighting and movement features I added, as they were challenging problems to tackle.
